Привет, амиго! В этой рубрике мы будем с тобой периодически делиться разными техническими секретами. Например, сегодня я покажу тебе, как пробраться в мобильник фирмы Motorola (метод работает с 2\3 апаратов) без ведома владельца. В жизни пригодится ;). Исходная ситуация: есть мобильный телефон, чтоб включить его, надо знать некий Unlock Code – сочетание нескольких цифр, которые живут в телефоне без связи с сим-картой (то есть, замена симки на свою и родную не помогает открыть телефон). А мы с тобой Unlock Code не знаем. Что делать будем? Наблюдай! Показываю только один раз ;)...

1.Так выглядит дисплей телефона, работающего в открытом режиме (без Lock).

2.Для того чтобы его залочить, заходим в меню и выбираем пункт Phone Setup.

3.Далее выбираем пункт Phone Lock..,

4.... в нем переходим на опцию Lock Now и нажимаем кнопку "Ок".

5.После нажатия кнопки "Ок" телефон переходит в закрытый режим. Теперь для доступа к меню, либо для набора номера необходимо ввести Unlock Code и перевести аппарат в открытый режим. Попытка ввести любой неверный код приводит к надписи Wrong Code.

6.Мы Unlock Code не знаем, поэтому будем хулиганить. Нажимаем клавишу "Решетка" и, не обращая внимания на надпись Wrong Code, удерживаем ее до тех пор, пока на дисплее не появится слово Test. Если оно благополучно появилось, значит, нам повезло, и данный аппарат оборудован тестовым режимом. Находясь в нем, можно узнать как пресловутый Unlock Code, так и Security Code, который позволяет сменить Unlock Code в любом режиме (даже если телефон уже залочен).

7.В тестовом режиме достаточно набрать "5", затем "9" (так, чтоб на дисплее появилась цифра 59) и нажать клавишу "Решетка" (она выполняет функции кнопки "Ввод"), чтоб телефон послушно выдал на дисплей свой Unlock Code.

8.Точно так же последовательное нажатие "5", "8" и "Решетки" приводит к выводу на дисплей Security Code.

9.Для выхода из тестового режима достаточно набрать "0", "1" и "Решетка" (а можно просто выключить и снова включить аппарат).

10.После ввода Unlock Code телефон переходит в открытый режим и начинает поиск сети.

11.Все, телефон открыт - можно пользоваться.

Вот такие пироги. Просто? Я тоже так думаю:). Мотай на ус. Увидимся в следующем номере :).
